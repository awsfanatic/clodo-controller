Clodo controller
================
Helps sending commands to [Clodo API](http://api.clodo.ru/).

Requirements
------------
* bash
* curl

Usage
-----
sh clodo-controller.sh <your_login> <your_API_key> <command> <server_id>

Example:
	sh clodo-controller.sh user@example.com 833957fba93b6b17cf38bd93e499f50d start 6
